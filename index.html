Bug description

Firefox will autofill the form, and fail to ask the user for a CVC. As far as the user can tell, all form elements have been filled, but on submit they'll receive an invalid CVC error.

Necessary conditions to reproduce the bug

  • Firefox (tested 102 ESR and 111 stable)
  • hidePostalCode: true
  • fontSize must be large enough and form width narrow enough
  • Credit card must be saved for autofill
